# 오늘의 식탁

주부를 위한 3일 단위 집밥 식단 추천 모바일 웹앱 MVP입니다.

## 실행 방법

1. Node.js를 설치합니다.
   - https://nodejs.org 에서 LTS 버전을 설치하면 됩니다.

2. 이 폴더에서 패키지를 설치합니다.

```bash
npm install
```

3. 개발 서버를 실행합니다.

```bash
npm run dev
```

4. 터미널에 표시되는 주소를 브라우저에서 엽니다.
   - 보통 `http://localhost:5173` 입니다.

## 포함된 기능

- 홈, 이번 주 식단, 장보기 3개 하단 탭
- 월·화·수 1차 장보기, 목·금·토 2차 장보기 구조
- 일요일 외식/자유식 카드
- 장보기 체크박스
- 체크 상태 localStorage 저장
- API 없이 더미데이터 기반 동작
